# OrðaTól 🐦‍⬛

**Old Norse:** *orð* (word) + *tól* (tool) — the word-tool.

A flashcard vocabulary app for **Norwegian on the Web (NoW) — Chapter 1**, built for iPad study.

## Live app

**[https://schneider128k.github.io/norwegian/](https://schneider128k.github.io/norwegian/)**

## Features

- 7 study units — 1K Ken, 1A Anna, 1M Maria, 1P Peter, Numbers, Phrases, All
- IPA pronunciation transcriptions from NTNU vocabulary tables
- Audio playback via Norwegian Web Speech API
- Etymology notes connecting words to English and German ancestors (toggleable)
- Spaced repetition (Again / Hard / Good / Easy) with SM-2-style intervals
- Print missed words after a practice session
- Progress saved in localStorage

## Source

Vocabulary from [Norwegian on the Web (NoW)](https://www.ntnu.edu/now) by NTNU.
